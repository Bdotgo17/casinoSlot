# Casino Slot (minimal)

This is a minimal browser-based slot machine demo that uses virtual coins only.

Important: For entertainment purposes only. No real money is taken or paid out. Virtual coins have no monetary value.

Run locally:

```bash
cd /Users/adam/casinoSlot-1
npm install   # no dependencies required but safe to run
npm start     # starts server on http://localhost:3131
```

Run smoke test:

```bash
npm test
```
