# 🎰 Casino Slot Machine

A browser-based casino slot machine game built with vanilla HTML, CSS, and JavaScript — no build tools required.

## How to Play

1. **Open `index.html`** in any modern web browser (Chrome, Firefox, Safari, Edge).
2. Use **− Bet** / **+ Bet** to adjust your wager (minimum 5 credits, steps of 5).
3. Press **SPIN** (or hit the **Spacebar**) to spin the reels.
4. Match symbols on the payline to win credits!
