# Casino Slot (minimal)

This is a minimal browser-based slot machine demo.

Run locally:

```bash
cd /Users/adam/casinoSlot-1
npm install   # none required but keeps habit
npm start     # starts server on http://localhost:3131
```

Run smoke test:

```bash
npm test
```
