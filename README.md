# Casino Slot (minimal)

This is a minimal browser-based slot machine demo that uses virtual coins only.

Important: For entertainment purposes only. No real money is taken or paid out. Virtual coins have no monetary value.

Run locally (production-like):

```bash
cd /Users/adam/casinoSlot-1
npm install
npm start     # starts server on http://localhost:3131
```

Run dev server with live reload:

```bash
cd /Users/adam/casinoSlot-1
npm install
npm run dev   # starts live-server on port 3131 and watches files
```

Run smoke test (starts server briefly and checks the page):

```bash
npm test
```

GitHub Pages deploy
- This repo includes a GitHub Actions workflow at `.github/workflows/gh-pages.yml` that publishes the repository root to GitHub Pages whenever `main` is pushed.
- If you want the site to be available at `https://<owner>.github.io/<repo>/`, enable Pages in the repository settings (or the action should create the `gh-pages` branch automatically).
